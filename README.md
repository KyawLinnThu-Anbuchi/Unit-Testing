
# NodeJS Unit Testing

NodeJS Unit Testing with Jest Testing Framework.

## Run Locally

Clone the project

```bash
  git clone https://github.com/KyawLinThuscm/NodeJS-Unit-Testing.git
```

To Run Test

```bash
  cd NodeJS-Unit-Testing
  npm install
  npm test
```
